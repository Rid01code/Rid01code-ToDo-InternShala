# ✅ React To-Do List App

This is a simple and elegant To-Do List application built with **React** using **Vite**. It allows users to add, edit, delete, and mark tasks as completed.

---

## 📚 Assignment Requirements Fulfilled

- ✅ Created with Vite (React project)
- ✅ Functional components: `App`, `Header`, `ToDoList`, `ToDoItem`
- ✅ State management using React `useState`
- ✅ Props passed between components correctly
- ✅ Dynamic list rendering using `.map()`
- ✅ Unique `key` for each to-do item
- ✅ Event handling: Add, Delete, Edit, Complete
- ✅ Clean styling with custom CSS
- ✅ GitHub repo includes `README.md` with setup instructions
